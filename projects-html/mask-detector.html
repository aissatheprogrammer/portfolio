Mask Detector

Description:

Mask Detector is an application designed to detect mask-wearing on individuals present at Simplon Grenoble in an image or video. This application utilizes a background classification algorithm that categorizes images into 2 categories:

  • Correct: proper mask-wearing
  • Missing: absence of a mask

Application Content:

Another face detection algorithm is used to determine if one or multiple faces are present in the image and perform classification on each detected face. It consists of 3 web pages, each offering different support for performing classification:

  • Upload File: classification on an uploaded image file (works with multiple individuals in a single image).
  • Take Picture: capturing a photo using the machine's camera and classifying the obtained image (works with a single individual).
